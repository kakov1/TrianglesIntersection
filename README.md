# TrianglesIntersection
Решение задачи о пересечении трёхмерных треугольников.
# Сборка проекта
```
git clone https://github.com/kakov1/TrianglesIntersection.git
cd TrianglesIntersection/
cmake -S . -B build/
cd build/
```
Запуск основной программы:
```
cd src/
make
./main
```
Запуск тестов:
```
cd tests/
make
./test
```
